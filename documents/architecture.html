3 · Personalization Score Rubric (LLM Judge)
Criterion Points Notes
Recipient's first name used correctly +0.30 Must appear verbatim in body
Property name mentioned naturally +0.25 Not just in opt-out suffix
At least one interest field used (city, amenity, or move timeline) +0.30 First match from profile fields
Each additional interest field beyond the first +0.075 each Capped at +0.15 bonus total
Full use of all profile fields ≥ 0.85 Minimum threshold for cases 1 & 2
Judge unavailable (no API key or exception) FAIL score = None → personalization_pass = False
